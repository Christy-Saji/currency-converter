# Currency Converter

A simple and user-friendly **Java Swing application** for converting currencies among major global economies. This project is designed as part of an academic course in Object-Oriented Programming.

## 📖 Overview

The **Currency Converter** allows users to easily convert between different currencies using hardcoded average exchange rates relative to the U.S. dollar. It features a graphical user interface built with Java Swing, offering a seamless experience with dropdowns, input fields, and real-time conversion results.

The application is perfect for travellers, finance students, and business professionals who need quick access to currency conversions without relying on internet connectivity.

---

## 🚀 Features

✔ Select source and target currencies from dropdown menus  
✔ Input any amount to convert  
✔ View the converted result instantly  
✔ Handles input errors gracefully with clear messages  
✔ Uses predefined exchange rates stored in a `HashMap` for fast lookups  
✔ Designed for scalability – easily add or update currencies and rates  
✔ Works offline without internet connection  

---

## 💻 Technologies Used

- **Java**  
- **Swing GUI Framework**  
- **HashMap** for storing exchange rates  
- **JComboBox, JTextField, JButton, JLabel** for interface elements
